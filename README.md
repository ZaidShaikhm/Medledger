# MedLedger

MedLedger is a responsive landing page for a pharma billing platform designed for retail pharmacies and wholesale distributors.

## Live Demo

Visit the deployed application:

https://medledger-azure.vercel.app/

## Tech Stack

* HTML
* CSS
* JavaScript
* Vite

## Prerequisites

Make sure you have:

* Node.js (v18 or later)
* npm

Check installed versions:

```bash
node -v
npm -v
```

## Installation

Clone the repository:

```bash
git clone https://github.com/ZaidShaikhm/Medledger.git
cd Medledger
```

Install dependencies:

```bash
npm install
```

## Run Locally

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

## Build for Production

```bash
npm run build
```

The production build will be generated in the `dist` folder.

## Preview Production Build

```bash
npm run preview
```

## Deployment

This project is deployed on Vercel:

https://medledger-azure.vercel.app/

Any changes pushed to the `main` branch will automatically trigger a new deployment on Vercel.
